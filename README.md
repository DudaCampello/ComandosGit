# ComandosGit :computer:
## Comandos para trabalhar no GIT BASH :clipboard:

- **cls** | LIMPA A TELA
- **tecla tab** | COMPLETA/FINALIZA UMA PALAVRA

- **ls** | LISTA ARQUIVOS
- **cd + nome pasta** | PARA ACESSAR A PASTA
- **cd** /  | PERMITE NAVEGAR
- **cd  ..**  | VOLTA 1 NÍVEL
- **dir** | PUXA TODAS AS PASTAS E ARQUIVOS
- **mkdir+nome**  | CRIA UMA PASTA
- **ls -a** | LISTA ARQUIVOS OCULTOS
- **rmdir nomepasta /s /q** | REMOVE PASTA

- **git init**  | INICIA UM REPOSITÓRIO. 
- **git add** | REALIZA A INCLUSÃO OU MODIFICAÇÃO DO ARQUIVO NO DIRETÓRIO LOCAL, PREPARANDO ELE PARA SER ENTREGUE.
- **git add  -**  | INFORMA QUE A INFORMAÇÃO SERÁ DIRECIONADA PARA O REPOSITÓRIO, MAS NÃO TRANSFERE
- **git reset <Nome do arquivo>** | DESFAZ O GIT ADD, em um arquivo (staging area)
- **git reset --mixed** | REVERTE A MUDANÇA EM UM BLOCO DE ARQUIVOS NOVOS E MODIFICADOS
- **git commit**  |
- **git commit -m "ex:commit inicial"** | CARREGA UMA MENSAGEM DE TEXTO(após git add -) 
- **git status**  | PARA CERTIFICAR QUE NÃO TEM NENHUMA PENDÊNCIA NO REPOSITÓRIO
- **git push original master**  | COMANDO PARA LEVAR DO REPOSITÓRIO LOCAL PARA O REMOTO
- **git push original main**  | COMANDO PARA LEVAR PARA NUVEM
- **git clone** | COLA O QUE FOI COPIADO (https://...)
- **git config  --list**  | DETALHA INFORMAÇÕES DO USUÁRIO
- **git config  --global user.e-mail "abc@abc.com"**  | CONFIGURAÇÃO DO USUÁRIO
- **git config  --global user.name Obama**  | CONFIGURAÇÃO DO USUÁRIO
- **git config  --global -ubset user.email**  | APAGA O E-MAIL DO USUÁRIO(CUIDADO)
- **git remote  -v**  | INFORMA A LISTA DE REPOSITÓRIO REMOTO CADASTRADO
